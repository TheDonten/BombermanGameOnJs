# BombermanGameOnJs
## Описание
 Игра представляет собой классический Bomberman на Nintendo NES.

## Что реализовано
  1. Анимация взрыва и бомб
  2. Проверка коллизий
  3. Разрушение обьектов
  4. Возможность играть по сети
